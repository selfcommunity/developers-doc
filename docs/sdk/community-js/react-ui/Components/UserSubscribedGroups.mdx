> The User Subscribed Groups Widget renders the list of the user groups.<br/>
This list is composed of items containing groups's image, title, info and a join/request button, that allows users to join or to request access to the group(based on group privacy).<br/>
The categories are shown in an orderly manner: few items are shown in the main card while the others can be displayed in a modal card by clicking on the "show all" button.

#### Usage

```jsx
 import React from 'react';
 import {SCContextProvider} from '@selfcommunity/react-core';
 import {UserSubscribedGroupsWidget} from '@selfcommunity/react-ui';

 function App() {
   const conf = {
     portal: '<community-path>',
     session: {
       type: 'OAuth',
       token: '<community-token>',
       refreshToken: '<community-refresh-token>',
       refreshTokenEndpoint: '<community-refresh-endpoint>',
     },
   };
   return (
     <SCContextProvider conf={conf}><UserSubscribedGroupsWidget /></SCContextProvider>
   );
 }
```
