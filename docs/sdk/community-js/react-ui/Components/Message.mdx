> The Message component renders a message item.<br/>
It can be rendered in different ways depending on its type that can be "snippet", "text", "document", "image" or "video".<br/>
Snippet messages contains the user avatar and username, the message content and the receipt date. The remaining message types <br/>
are self-explanatory and contain the body of the message related to their type.

#### Usage

```jsx
 import React from 'react';
 import {SCContextProvider} from '@selfcommunity/react-core';
 import {Message} from '@selfcommunity/react-ui';

 function App() {
   const conf = {
     portal: '<community-path>',
     session: {
       type: 'OAuth',
       token: '<community-token>',
       refreshToken: '<community-refresh-token>',
       refreshTokenEndpoint: '<community-refresh-endpoint>',
     },
   };
   return (
     <SCContextProvider conf={conf}><Message /></SCContextProvider>
   );
 }
```
