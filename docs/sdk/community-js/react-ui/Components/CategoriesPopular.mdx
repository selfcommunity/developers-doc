> The Categories Popular Widget renders a list of popular categories.<br/>
This list is composed of items containing category's image, title, info and a follow button, that allows users to follow/unfollow a category.<br/>
The categories are shown in an orderly manner: few items are shown in the main card while the others can be displayed in a modal card by clicking on the "show all" button.

import {CategoriesPopularWidget} from '@selfcommunity/react-ui';

#### Usage

```jsx
 import React from 'react';
 import {SCContextProvider} from '@selfcommunity/react-core';
 import {CategoriesPopularWidget} from '@selfcommunity/react-ui';

 function App() {
   const conf = {
     portal: '<community-path>',
     session: {
       type: 'OAuth',
       token: '<community-token>',
       refreshToken: '<community-refresh-token>',
       refreshTokenEndpoint: '<community-refresh-endpoint>',
     },
   };
   return (
     <SCContextProvider conf={conf}><CategoriesPopularWidget /></SCContextProvider>
   );
 }
```
