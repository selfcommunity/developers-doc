:::info
This custom hook is used to fetch a tag object.
:::

## API

`useSCFetchTag(id, tag) => {scTag, setSCTag}`


### Arguments

1. `id` (number): The tag Id.
2. `tag` ([SCTagType](../Types/tag)): The tag obj.


### Returns 

`{scTag, setSCTag}`:
